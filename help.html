Help - Olex2RTZ

Welcome to the help page for Olex2RTZ. Here you will find instructions on how to use the application:

  • Uploading a file: Go to the main page and upload an olexplot.gz file to analyze.
  • Viewing routes: After uploading, you can view the routes and waypoints on the map.
  • Converting routes: Select a route from the dropdown menu and click "Convert" to download the RTZ file.

If you encounter any issues, please contact support at support@olex2rtz.com.
