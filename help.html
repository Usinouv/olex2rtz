Help - Olex2RTZ

Exporting Routes from Olex

  1. Click on Settings then Save data to storage device.

The Select device to save to window should open.

  1. Click on the device you want to save to.

The file browser should open.

Note: Olex requires the device to be formatted in FAT32.

  1. Navigate to the backup directory, then click on All plotter data.
  2. Click Save.

Using Olex2RTZ

  1. Go to the main page of the website.
  2. Upload your olexplot.gz file by clicking on Analyze the file.
  3. After upload, you will see all routes and waypoints extracted from your file.
  4. Select a route from the dropdown menu to preview it on the map.
  5. If needed, you can rename the route before exporting.
  6. Click on Convert to download the route in RTZ format.

Note: The system automatically detects waypoints and organizes them correctly into a route.

If you encounter any issues, please contact adhont.
